## 项目介绍
通过接口自动化模拟i茅台app实现每日自动预约茅台酒的功能,可添加多用户,选择本市出货量最大的门店,或预约你的位置附近门店

软件会在指定时间开始对管理的用户进行批量预约。




本程序是对该项目(https://github.com/lisongkun)的WPF 使用blazor客户端实现
